ขอแก้ไขเพิ่มเติม ยังไม่เอา api key จาก console.anthropic.com ครับ แต่จะให้แบ่งการทำงานให้ชัดเจนดังนี้
- admin page 
  - คือ Line id : "Oady" ที่จะจัดการทั้งหมดได้ รวมถึง เพิ่มข้อสอบ pdf ไฟล์ต่างๆ เข้าไป
  - ให้มีปุ่มลัด
- ผู้ปกครอง page
  - จัดการ ตั้งชื่อ , รูปภาพนักเรียน เก็บข้อมูลต่างๆ ไว้ที่ google sheet ได้ หลังจาก full version
  - มีปุ่มที่กดเข้าไปเพื่อเปลี่ยนโหมด "ผู้ปกครอง" ที่มุมขวาบน

- นักเรียน page
  - ปรับให้เหมาะสม สำหรับนักเรียน 

- ทำให้ dashboard เห็นข้อมูลชัดเจน เพื่อวางแผนให้ลูก
- ทำเปรียบเทียบ ระหว่าง ทดลองใช้ , full version ให้ชัดเจน
- ขั้นตอนจ่ายเงิน ให้แนบสลิป ธนาคาร และแจ้งเตือน admin 
- หลังแจ้งเตือน admin แล้ว admin สามารถกด approve ได้จากหน้า admin page 
- ให้ backup ข้อมูลให้อัตโนมัติ อย่างน้อยอาทิตย์ละครั้ง หรือสามารถ กดปุ่ม backup ได้เอง จากหน้า "Setting" ผู้ปกครอง

-----------------------------
ตรวจเช็คความถูกต้องที่ให้ run sql จากภาพทั้ง 2 ว่าถูกต้องไหม
เวลากด backup ข้อมูลเข้า google sheet ให้เพิ่มอีก column เพื่อตรวจสอบลูกค้าแต่ละคนได้ กันซ้ำกัน
ให้พาตรวจเช็ค รหัสอีกครััง step by step กันพลาด ทั้ง admin page , ผู้ปกครอง page
version ทดลองให้เพิ่มข้อสอบเป็นอย่างละ 10 ข้อ
โหมดผู้ปกครองให้สามารถเปลี่ยนรูปและชื่อนักเรียนได้เอง
ปรับขนาด fonts ให้เล็ก-ใหญ่ได้ 4 ระดับ
กดรหัส full version ไม่มีอะไรเกิดขึ้น
กด backup ข้อมูล ไม่มีอะไรเกิดขึ้น
ตรวจเช็ค ข้อสอบที่เอามาให้ใช้ว่าถูกโรงเรียน ตรงกับปีที่จะทำการทดสอบด้วยไหม

---------------------------------------
ที่ หน้าผู้ปกครอง ให้สามารถ ดึงข้อสอบใหม่ๆ จาก server มาเพิ่มได้ โดยที่
  - ทดลองใช้ ดึงเพิ่มได้ 1 ครั้ง และทำข้อสอบได้ 10 ข้อ 
  - เปลี่ยนชื่อ , ใส่รูป ลูก, ผู้ปกครองได้ง่าย และน่ารัก ขยายขนาดรูปให้ใหญ่ขึ้น
  - มีปุ่ม เพื่อขอแอด line oa เพื่อจะได้ทำการรับแจ้งเตือน หรือติดต่อ admin ผ่าน line oa ได้ง่าย
  - ตรวจเช็คปุ่ม backup ใช้งานได้ปกติไหม 
  - ผู้ปกครอง สามารถเลือก เพิ่ม, ลบ โรงเรียนมาให้ลูกฝึกทำได้
  - ปรับ dashboard ให้ผู้ปกครองได้ดูข้อมูล responsive ทั้ง phone, tablet , laptop

  ----------------------------------------
  อยากรู้ว่าการ backup ข้อมูล ได้แบ่งชุด data ของลูกค้าแต่ละคนไว้ด้วยไหม ถ้ายังช่วยเพิ่มให้ด้วย ทั้งที่ sql , google sheet
  อยากให้มีปุ่ม restore ข้อมูลเพิ่ม สำหรับ ผู้ปกครอง , admin ระบุ restore ให้ลูกค้าแบบเลือกได้
  ใน google sheet ตอนนี้มีข้อมูลตามภาพที่ส่งไป ช่วย update ให้ด้วยครับ ถ้ายังไม่เพิ่ม column CustomerID ให้เพิ่ม เพื่อง่ายต่อการจัดการทีหลัง
  อยากให้สามารถ กด cancel ถ้ากด เริ่มทำข้อสอบไปแล้ว ต้องการ หยุด
  จากรูปที่ส่งไป import pdf file ไม่ได้ขึ้น error ช่วยแก้ไข หาวิธี จัดการไฟล์ pdf แบบสแกนให้ด้วย
  ที่หน้า "หน้าเพิ่มข้อสอบ" , "นำเข้า pdf" ให้เพิ่มปุ่มที่สามารถเพิ่ม โรงเรียน, วิชา, ปี ได้เองทีหลัง
  ที่หน้า "หน้าเพิ่มข้อสอบ" ให้สามารถใส่รูปภาพ ที่โจทย์,คำอธิบายเฉลย ,ตัวเลือกได้ โดยการ copy & paste , หรือเลือกไฟล์รูปภาพในเครื่อง   
  ยังไม่เห็นว่า ผู้ปกครองสามารถ upload รูปได้เอง ที่เป็น jpg file เพิ่มให้ด้วยครับ อาจจะระบุขนาดไม่เกิน 1 Mb. เพือประหยัดพื้นที่
  คำถามสุดท้าย จากระบบตอนนี้ เป็น free ที่ supabase อยู่ แต่อยากรู้ จะหมด version free เร็วไหม เมื่อไหร่

  รุปนักเรียน ที่ upload เอง ให้สามารถปรับแต่งรูป resize ได้
  "🔓 โหมดผู้ปกครอง · กดรูปเพื่อแก้ข้อมูลนักเรียน" กดรูป ไม่มีอะไรเกิดขึ้น
  ไม่เห็น ปุ่ม restore 
  เพิ่มหน้ารายงาน ให้ save เป็น ขนาดหน้า a4 ไฟล์ jpg เพื่อส่งต่อได้เลย
  ในหน้ารายงาน ให้ใส่ข้อมูลให้เหมาะสม
  จะตรวจเช็คยังไงว่า คนไหนเป็น ผู้ปกครอง , ลูก  ถ้าเปิดคนละ device ช่วยตรวจสอบและแก้ไขให้ถูกต้อง
  เพิ่มวันสำหรับ full version ให้ใช้งานได้เป็น 45 วัน
  Restore ข้อมูล มีหลายแบบ อยากได้แบบที่ 1 ,2 
  ทำรูปภาพในข้อสอบด้วยครับ ให้ได้ทั้ง โจทย์ ,ตัวเลือก ,คำเฉลย หรือกดดู pdf ได้เลย
